An Efficient Blockchain-Based Framework for File Sharing

This project focuses on creating a secure, decentralized file-sharing system using blockchain technology. Traditional file-sharing systems often suffer from issues like data tampering, single points of failure, and lack of transparency. Our solution eliminates these risks by leveraging smart contracts and distributed ledger features of blockchain.

The system allows users to upload and share files via blockchain nodes, where each file transaction is securely recorded. Access controls are managed using smart contracts to ensure only authorized users can download or view shared content.

Key Features:

  • Decentralized file storage and access
  • Immutable transaction history
  • Secure file sharing with pre hashing

Technologies Used: Python,flask

Outcome: Provided a tamper-proof and transparent file-sharing framework, demonstrating strong potential for real-world use in industries needing secure digital document exchange.
